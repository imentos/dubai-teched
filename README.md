# Smart City & Traffic Control powered by Deep Learning

## How to setup:
1. Go to ```root``` folder:
  ```
  npm install
  ```
  
2. Go to ```web``` folder
  ```
  bower install
  ```
  
## How to start:
1. Go to ```root``` folder:
  ```
  node index.js
  ```
  
2. Go to [http://localhost:3000/index.html](http://localhost:3000/index.html) in Chrome
